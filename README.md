tenhou-analysis
===============

Tenhou analysis tool for happy mahjong players.

How to Use
----------

```
$ git clone https://github.com/noraesae/tenhou-analysis.git
$ cd tenhou-analysis
$ npm install
$ node index.js
```

*It needs redis-server to run the script.*

You can access the statistic page with `http://localhost:2331`.
